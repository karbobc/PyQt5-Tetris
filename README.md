
## PyQt5-Tetris

### PyQt5俄罗斯方块，PyQt5-Tetris，基于PyQt5写的一个小游戏

####  💻 开发环境

- Windows 10

- Python 3.8.2

#### 📦 安装依赖

```shell
pip install PyQt5
```

#### 📝 功能

- 🎵 播放音乐
- ⏸︎ 暂停游戏
- 🎸 消除方块的音效

#### ✅ Todo

- 📄 游戏存档
- 🖱️ 窗口拖拽


#### ⌨️ 使用方法

1、 直接运行`main.py`文件

2、打开终端，进入到项目目录下执行

```
python main.py
```

#### ❗ 已知BUG

- 点击按钮后, 键盘事件会失去监听，需要再次点击游戏界面才可以恢复（希望大佬们可以提供解决方案）
